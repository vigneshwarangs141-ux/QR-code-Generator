QR Code Generator

This is a simple QR Code Generator built using HTML, CSS, and JavaScript.

It allows users to enter any text and instantly generate a QR code.

Features

Generate QR code from user input

Simple and clean design

Smooth QR image display

No backend required

Technologies Used

HTML

CSS

JavaScript

│
├── index.html

How to Use

Open the project folder.

Open index.html in your browser.

Enter text in the input field.

Click the Generate Code button.

The QR code will appear below.

How It Works

The user enters text in the input box.

When the button is clicked, JavaScript checks if the input is not empty.

The QR code image is generated and displayed on the page.
